## Soil Survivors DLC Game Mode

Nearly single-handedly designed and built '[Soil Survivors](https://youtu.be/rHGN1rEQWNU?t=18)' game mode in 3 months, blending Counter-Strike elimination with PUBG shrinking zones. Scripted game mode systems including logic, UI, server configuration, and bot balancing using minimal development resources while maintaining high quality standards.

## Solo Ops Companions

Built AI companion system for '[Solo Ops](https://youtu.be/wPaWBYMQ5Fw)' enabling single players to experience cooperative gameplay. Designed intelligent companion selection UI and modified AI behavior trees to prioritize player support and base defense, creating seamless solo-to-multiplayer experience that increased player engagement.

## Infinity Time End Game Mode

Architected '[Infinity Time](https://youtu.be/DCBprjJiPJY)' end-game mode featuring mech vs gnome army combat. Designed complete enemy ecosystem including boss behaviors, AI systems, and unique attack patterns. Collaborated with avfx & animation teams to deliver cohesive combat experience that became the game's end-game content.
